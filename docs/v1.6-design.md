# Deck v1.6 — Design

Activity feed, session bookmarks, and git blame integration.

## Pillars

### Pillar 1: Activity Feed
Real-time-style feed of all Claude Code activity across projects.

- `/activity` page showing a chronological feed of events
- Event types: session started, session ended, high-cost session, new project detected, long session alert
- Each event: icon, description, project badge, timestamp, link to session
- Filterable by project and event type
- Auto-refreshes every 30 seconds
- Compact timeline format with relative timestamps

**Files:** `src/app/activity/page.tsx` (new), `src/app/api/activity/route.ts` (new), update sidebar, update command palette.

### Pillar 2: Session Bookmarks
Bookmark specific messages within sessions for quick reference.

- Bookmark icon on each message in session detail view
- `/bookmarks` page showing all bookmarked messages across sessions
- Each bookmark: session link, message preview, timestamp, project
- Stored in `~/.deck/bookmarks.json`
- Search/filter bookmarks by keyword
- Remove bookmark with one click

**Files:** `src/app/bookmarks/page.tsx` (new), `src/app/api/bookmarks/route.ts` (new), `src/lib/bookmarks.ts` (new), update session detail page.

### Pillar 3: Git Integration Dashboard
Deeper git insights linked to Claude Code sessions.

- `/git` page showing git activity correlated with sessions
- Recent commits across all projects with session linkage
- Commit frequency chart (daily bars for last 30 days)
- Files most frequently modified by Claude sessions
- Branch activity: active branches per project

**Files:** `src/app/git/page.tsx` (new), `src/app/api/git/route.ts` (new), update sidebar, update command palette.
