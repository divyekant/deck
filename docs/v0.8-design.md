## Pillars

### Pillar 1: Repo Pulse
Per-repository health/activity dashboard — like GitHub's Pulse view but for Claude Code work.

- `/pulse` page showing all tracked repos with activity indicators
- For each repo: recent git commits (last 7 days), branches, files most frequently touched in sessions, session count trend (sparkline)
- Color-coded activity level: hot (daily sessions), warm (weekly), cold (no recent activity)
- Click repo → goes to existing `/repos/[name]` deep-dive
- Summary stats at top: total commits this week, most active repo, repos with no activity

**Files:** `src/app/pulse/page.tsx` (new), `src/app/api/pulse/route.ts` (new), update sidebar, update command palette.

### Pillar 2: Work Graph
Visual network showing relationships between sessions, projects, and files.

- `/work-graph` page with SVG-based node graph
- Three node types: Projects (large colored circles), Sessions (medium dots), Files (small dots)
- Edges connect sessions to files they modified (using diffs data)
- Projects cluster their sessions visually
- Layout: force-directed style using simple positioning algorithm (no external lib)
- Hover a node → highlight its connections
- Click project node → navigate to `/repos/[name]`
- Click session node → navigate to `/sessions/[id]`
- Filter by date range (last 7d / 30d / 90d)

**Files:** `src/app/work-graph/page.tsx` (new), `src/app/api/work-graph/route.ts` (new), update sidebar, update command palette.

### Pillar 3: Snapshots
Auto-generated session digests — condensed summaries of what happened in each session.

- `/snapshots` page showing a grid of session snapshot cards
- Each snapshot auto-extracts from session data: key decisions (from assistant messages), files modified (from tool calls), commands run (from Bash tool calls), total tokens/cost
- Compact card format: project badge, timestamp, 2-3 bullet points of key actions, file count badge, cost badge
- Click card → full snapshot detail with all extracted info
- Filter by project, sort by date or cost
- Search within snapshots

**Files:** `src/app/snapshots/page.tsx` (new), `src/app/api/snapshots/route.ts` (new), update sidebar, update command palette.
