# Deck v1.2 — Design

Session replay and prompt library.

## Pillars

### Pillar 1: Session Replay
Step through a session's conversation message by message, like a replay.

- `/sessions/[id]/replay` page
- Shows messages one at a time with forward/back controls
- Progress bar showing position in conversation
- Auto-play mode with configurable speed (1s, 2s, 5s per message)
- Each message appears with a typing animation effect
- Stats update live as you progress (running cost, token count)

**Files:** `src/app/sessions/[id]/replay/page.tsx` (new).

### Pillar 2: Prompt Library
Save and browse effective prompts extracted from sessions.

- `/prompts` page showing saved prompts
- Auto-extract first prompts from all sessions, deduplicate similar ones
- Each prompt card: text, project it was used in, model, result metrics (cost, duration)
- Search/filter prompts
- Group by category (inferred from content: "bug fix", "feature", "refactor", etc.)
- Sort by: most used, most recent, lowest cost

**Files:** `src/app/prompts/page.tsx` (new), `src/app/api/prompts/route.ts` (new), update sidebar, update command palette.

### Pillar 3: Session Tags Analytics
Analyze session tags to show patterns.

- New section on analytics page: "Tags Overview"
- Tag cloud visualization (size proportional to usage count)
- Tags breakdown: which tags are most used, avg cost per tag, avg duration per tag
- Tags trend: how tag usage changes over time

**Files:** `src/components/tags-analytics.tsx` (new), update `src/app/analytics/page.tsx`.
