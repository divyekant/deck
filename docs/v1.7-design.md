# Deck v1.7 — Design

Dashboard customization, data export, and session annotations.

## Pillars

### Pillar 1: Dashboard Customization
Let users choose which widgets appear on the home page.

- Settings panel on home page (gear icon) to toggle widgets on/off
- Widgets: Stats Row, Activity Chart, Cost Trend, Budget Gauge, Work Hours, Cost by Model, Streak, Highlights, Favorites, Daily Digest, Recent Sessions
- Preferences stored in `~/.deck/dashboard-prefs.json`
- Default: all widgets visible
- Compact grid layout respects hidden widgets

**Files:** `src/lib/dashboard-prefs.ts` (new), `src/app/api/dashboard-prefs/route.ts` (new), update `src/app/page.tsx`.

### Pillar 2: Data Export
Export session data and analytics in CSV/JSON formats.

- `/export` page with export configuration
- Export types: All Sessions, Cost Report, Token Usage, Model Usage
- Format selector: JSON or CSV
- Date range filter
- Download button that triggers browser download
- Preview of first 5 rows before download

**Files:** `src/app/export/page.tsx` (new), `src/app/api/export/route.ts` (new), update sidebar, update command palette.

### Pillar 3: Session Annotations
Add notes and tags to individual sessions.

- Annotation panel on session detail page (collapsible)
- Add text notes with timestamps
- Add/remove custom tags (beyond auto-detected ones)
- Stored in `~/.deck/annotations.json`
- Annotations visible in session list as small icons
- Tag filter on sessions page uses both auto + custom tags

**Files:** `src/lib/annotations.ts` (new or update existing), `src/app/api/annotations/route.ts` (new), `src/components/annotation-panel.tsx` (new), update session detail page.
