# Deck v0.7 — Design

Continuing to close the gap with the reference dashboard. Remaining sidebar items: Timeline, Diffs, Snapshots, Work Graph, Repo Pulse, Skills, Ports.

## Pillars

### Pillar 1: Timeline
Vertical chronological feed of all sessions grouped by day. An alternative to the table view — more visual, more context at a glance.

- `/timeline` page with infinite-scroll-style vertical timeline
- Sessions grouped by day with date headers ("Today", "Yesterday", "Feb 25", etc.)
- Each session as a rich card: colored left border (project color), project badge, model badge, first prompt, cost, duration, message count, timestamp
- Day summary at each date header: "5 sessions, $12.40"
- Compact mode toggle (show/hide prompt text for density)

**Files:** `src/app/timeline/page.tsx` (new), update sidebar, update command palette.

### Pillar 2: Diffs — Session File Changes
Parse session tool calls to show what files were modified during each session.

- New `/api/sessions/[id]/diffs` endpoint that parses tool_use blocks (Edit, Write, Bash) from the session to extract file paths
- Show file change summary on session detail page: list of modified files with badges (created/edited)
- New `/diffs` page: aggregate view of all file changes across recent sessions, grouped by file — "which files are touched most?"

**Files:** `src/app/api/sessions/[id]/diffs/route.ts` (new), `src/app/diffs/page.tsx` (new), update `src/app/sessions/[id]/page.tsx`, update sidebar.

### Pillar 3: Skills Browser
Browse and view Claude Code custom slash commands and project CLAUDE.md files.

- `/skills` page that scans `~/.claude/commands/` for custom commands (markdown files)
- Shows each command: name, description (first line), full content expandable
- Also scans project CLAUDE.md files from known project directories
- Read-only view — useful for understanding what's configured

**Files:** `src/app/skills/page.tsx` (new), `src/app/api/skills/route.ts` (new), update sidebar.
