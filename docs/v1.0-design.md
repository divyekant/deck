# Deck v1.0 — Design

Major milestone. Focus on polish, performance, and UX refinements across the entire app.

## Pillars

### Pillar 1: Global Keyboard Navigation
Full keyboard navigation across all pages.

- `Keyboard` component that registers global shortcuts:
  - `g h` → Home, `g s` → Sessions, `g a` → Analytics, `g c` → Costs, `g l` → Live, `g r` → Repos, `g t` → Timeline, `g p` → Pulse
  - `?` → Show keyboard shortcuts help modal
  - `Cmd+K` already opens command palette (keep existing)
- Shortcuts help modal listing all available shortcuts
- Small `?` button in bottom-right corner to open help

**Files:** `src/components/keyboard-nav.tsx` (new), `src/components/shortcuts-modal.tsx` (new), update layout.

### Pillar 2: Dashboard Widgets — Streak & Highlights
Add two new home page widgets for engagement.

- **Streak Widget**: Current coding streak (consecutive days with sessions), longest streak, streak calendar (last 30 days as a grid of dots)
- **Highlights Widget**: Auto-generated daily highlights — "Today: 5 sessions, $4.20 spent, 3 projects touched, most used model: claude-sonnet-4-5"

**Files:** `src/components/streak-widget.tsx` (new), `src/components/highlights-widget.tsx` (new), update `src/app/page.tsx`.

### Pillar 3: Session Detail Polish
Enhance the session detail page experience.

- **Message timestamps**: Show relative timestamps on each message
- **Copy button**: Copy message text to clipboard
- **Jump to top/bottom**: Floating buttons for long conversations
- **Message count indicator**: "Message 5 of 42" as you scroll
- **Collapsible thinking blocks**: Thinking blocks start collapsed with "Show thinking" toggle

**Files:** Update `src/app/sessions/[id]/page.tsx`, update `src/components/message-view.tsx`.
