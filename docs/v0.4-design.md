# Deck v0.4 — Design

## Pillars

### Pillar 1: Project Deep-Dive
Turn the Repos page from a summary grid into a real analytics view. Clicking a project card navigates to `/repos/[name]` with:
- Project-scoped stats: total sessions, total cost, models used, active days
- Project-scoped cost trend chart (reuse CostTrendChart component)
- Project-scoped session list with all existing filters (search, date, bookmarks)
- Top sessions by cost (quick way to find expensive runs)

**Files:** `src/app/repos/[name]/page.tsx` (new), `src/app/api/projects/[name]/route.ts` (new), update `src/app/repos/page.tsx` to link cards.

### Pillar 2: Session Annotations
Add lightweight metadata per session — tags and notes — persisted to `~/.deck/annotations.json`.
- Tag chips on session cards and detail view (add/remove inline)
- Free-text note field on session detail view (auto-saves on blur)
- Filter sessions by tag on the sessions list page
- Pre-defined quick tags: "bug-fix", "feature", "refactor", "exploration", "review"
- Custom tags (type to create)

**Files:** `src/lib/annotations.ts` (new), `src/app/api/annotations/route.ts` (new), update `src/components/session-card.tsx`, update `src/app/sessions/page.tsx`, update `src/app/sessions/[id]/page.tsx`.

### Pillar 3: Collapsible Sidebar + Polish
- Sidebar collapse toggle (icon-only mode, persisted to settings)
- Debounce search inputs (300ms)
- Clean up remaining `any` types in tool-call-view.tsx
- Add model filter dropdown to sessions list (currently only search + date + bookmarks)
- Improve session detail header layout (currently wraps awkwardly on narrow screens)

**Files:** update `src/components/sidebar.tsx`, update `src/app/sessions/page.tsx`, update `src/components/tool-call-view.tsx`, update `src/app/sessions/[id]/page.tsx`.
