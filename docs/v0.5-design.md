# Deck v0.5 — Design

## Pillars

### Pillar 1: Full-Text Search
Search across all session message content, not just the first prompt.

- New `/search` page with search input, results list with highlighted matches
- New `/api/search?q=` endpoint that scans session JSONL files for matching text blocks
- Results show: session ID, project, model, matched text snippet (with query highlighted), timestamp
- Click result → navigates to session detail
- Add "Search" to sidebar nav under Overview section
- Register search in command palette

Implementation: Iterate all sessions via `listSessions()`, for each result fetch the session detail and search through text/thinking blocks. Return top 50 matches. Cache-friendly since sessions are append-only.

**Files:** `src/app/search/page.tsx` (new), `src/app/api/search/route.ts` (new), update sidebar, update command palette.

### Pillar 2: Analytics Page
Dedicated `/analytics` page with deeper insights than the home dashboard.

- Weekly cost breakdown: bar chart showing cost per week for the last 12 weeks
- Model efficiency: table comparing models by avg cost/session, avg tokens/session, total spend
- Project ranking: top 10 projects by cost with bar visualization
- Session duration trends: avg session duration over time
- Peak usage hours: enhanced version of the work-hours heatmap

**Files:** `src/app/analytics/page.tsx` (new), `src/app/api/analytics/route.ts` (new), update sidebar nav.

### Pillar 3: Bulk Operations
Multi-select sessions for batch actions.

- Checkbox column on sessions table (select individual or select all visible)
- Bulk action bar appears when sessions are selected: Export CSV, Export Markdown, Add Tag, Bookmark All
- CSV export: downloads a CSV with columns: id, project, model, firstPrompt, messageCount, tokens, cost, date
- Markdown export: downloads a zip file containing individual markdown files per session
- Bulk tag: opens a tag picker to add a tag to all selected sessions
- Bulk bookmark: toggles bookmark on all selected sessions

**Files:** update `src/app/sessions/page.tsx`, new `src/app/api/export/route.ts`, new `src/lib/csv.ts`.
