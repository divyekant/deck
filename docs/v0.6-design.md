# Deck v0.6 — Design

Reference: Screenshot of similar dashboard with richer nav, project colors, live sessions, costs page, commits tracking, dynamic greeting.

## Pillars

### Pillar 1: Project Colors
Assign each project a consistent color (hash-based). Apply everywhere:
- Colored dot before project name in session table rows
- Colored left border on session cards (home page recent sessions)
- Colored accent on repo cards
- Colored bars in Cost by Model breakdown

**Files:** `src/lib/project-colors.ts` (new), update `src/components/session-card.tsx`, update `src/app/sessions/page.tsx`, update `src/app/repos/page.tsx`, update `src/components/cost-breakdown.tsx`.

### Pillar 2: Live Sessions + Costs Pages
Two new pages matching the reference dashboard.

**Live page (`/live`):**
- Shows currently running sessions with real-time status
- Auto-refreshes every 5s
- Each running session shows: project name (colored), model, prompt, elapsed time, live token count
- Pulsing green indicator for active sessions
- Empty state when nothing is running

**Costs page (`/costs`):**
- Dedicated cost analysis view
- Monthly cost total with daily breakdown chart
- Cost by model (horizontal colored bars with dollar amounts — match reference style)
- Cost by project (top 10)
- Most expensive sessions table (top 20)
- Date range selector (this month, last month, last 90 days, all time)

**Files:** `src/app/live/page.tsx` (new), `src/app/costs/page.tsx` (new), `src/app/api/costs/route.ts` (new), update sidebar, update command palette.

### Pillar 3: Enhanced Home Page
Match the reference's richer dashboard:
- **Dynamic greeting** with insights: "You have X repos set up. Y has the most sessions. You've spent $Z today."
- **Commits Today stat** — count commits across all project dirs using git log

- **Better Cost by Model** — horizontal colored bars (not just a list) matching the reference screenshot
