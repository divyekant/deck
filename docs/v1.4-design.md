# Deck v1.4 — Design

Project health dashboard and export/reporting.

## Pillars

### Pillar 1: Project Health Dashboard
Aggregated health score and metrics for each project.

- `/health` page showing project health cards
- Health score (0-100) based on: session frequency, cost efficiency, cache hit rate, average session length
- Trend arrow: improving/declining/stable based on last 7 days vs previous 7 days
- Health breakdown: radar/spider chart showing the 4 health dimensions
- Sort by: healthiest, most active, most expensive

**Files:** `src/app/health/page.tsx` (new), `src/app/api/health/route.ts` (new), update sidebar, update command palette.

### Pillar 2: PDF Report Generation
Generate downloadable PDF-style reports of session activity.

- `/reports` page with report configuration
- Report types: Weekly Summary, Monthly Summary, Project Report
- Report preview as styled HTML (printable via browser print)
- Includes: activity chart, cost breakdown, top sessions, model usage
- "Print Report" button that triggers window.print() with print-optimized CSS

**Files:** `src/app/reports/page.tsx` (new), `src/app/api/reports/route.ts` (new), update sidebar, update command palette.

### Pillar 3: Favorites & Quick Access
Pin frequently used pages and sessions for quick access.

- Favorites bar below the greeting on the home page
- Pin/unpin sessions, projects, or pages
- Stored in `~/.deck/favorites.json`
- Quick access row: small cards showing pinned items with one-click navigation
- "Pin" button on session detail, repo detail, and other pages

**Files:** `src/lib/favorites.ts` (new), `src/app/api/favorites/route.ts` (new), `src/components/favorites-bar.tsx` (new), update home page.
