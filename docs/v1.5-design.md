# Deck v1.5 — Design

Notifications, daily digest, and session templates.

## Pillars

### Pillar 1: Notifications & Alerts
Budget alerts and activity notifications.

- `/notifications` page showing notification history
- Alert types: budget threshold (80%, 90%, 100%), long-running session (>30min), daily cost spike (2x average)
- Notifications computed on page load from session data (no push notifications)
- Notification bell icon in sidebar with unread count badge
- Each notification: type icon, message, timestamp, dismiss button
- Stored in `~/.deck/notifications.json` (dismissed state)

**Files:** `src/app/notifications/page.tsx` (new), `src/app/api/notifications/route.ts` (new), `src/lib/notifications.ts` (new), update sidebar.

### Pillar 2: Daily Digest
Auto-generated daily summary card on the home page.

- Summary of yesterday's activity: sessions, cost, top project, most used model, interesting stats
- "Yesterday you had X sessions across Y projects. You spent $Z, mostly on [project]. Your cache hit rate was A%."
- Expandable detail section with full breakdown
- Shows comparison to daily average

**Files:** `src/components/daily-digest.tsx` (new), update `src/app/page.tsx`.

### Pillar 3: Session Templates
Pre-defined session starters for common tasks.

- `/templates` page showing available templates
- Each template: name, description, prompt text, suggested model, category
- "Launch" button that navigates to `/sessions/new` with the prompt pre-filled
- Default templates: "Bug Fix", "Code Review", "Feature Build", "Refactor", "Exploration", "Documentation"
- Custom template creation form

**Files:** `src/app/templates/page.tsx` (new), `src/app/api/templates/route.ts` (new), `src/lib/templates.ts` (new), update sidebar, update command palette.
