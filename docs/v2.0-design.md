# Deck v2.0 — Design

Major release: About/changelog page, onboarding experience, and global status bar.

## Pillars

### Pillar 1: About & Changelog
Version info and feature changelog for the app.

- `/about` page showing:
  - Deck logo and version number (v2.0)
  - "A local-first dashboard for Claude Code analytics"
  - Feature list organized by version (v0.1 through v2.0)
  - Each version: version number, title, release features (bullet list)
  - Tech stack section: Next.js, React, TypeScript, Docker, inline SVG charts
  - Link to GitHub repo
  - Footer with attribution

**Files:** `src/app/about/page.tsx` (new), update sidebar, update command palette.

### Pillar 2: Onboarding
First-time user experience when no sessions exist.

- Welcome screen on the home page when zero sessions detected
- Step-by-step getting started guide:
  1. "Install Claude Code CLI"
  2. "Run a few sessions"
  3. "Deck will automatically detect your sessions"
- Animated illustration (CSS-only) showing data flowing in
- "Explore Demo" button that links to the main pages
- Dismissible — stores `onboarded` flag in localStorage

**Files:** `src/components/onboarding.tsx` (new), update home page.

### Pillar 3: Global Status Bar
Persistent status bar at the bottom showing real-time system status.

- Fixed bottom bar (above mobile nav on small screens)
- Shows: active session count (live dot), total sessions today, today's cost, last sync time
- Updates every 30 seconds
- Compact: single row, small text
- Collapsible on mobile (just shows the live dot)

**Files:** `src/components/status-bar.tsx` (new), update layout.
