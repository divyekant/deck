# Deck v1.8 — Design

Search enhancement, session grouping, and cost forecasting.

## Pillars

### Pillar 1: Enhanced Search
Full-text search across session conversations with highlighted results.

- Upgrade `/search` page with full-text message content search
- Search results show: session title, matching message snippet with highlighted keyword, project, date
- Results grouped by session with match count badge
- Paginated results (20 per page)
- Search debounced at 300ms

**Files:** `src/app/api/search/route.ts` (new or update), `src/app/search/page.tsx` (update).

### Pillar 2: Session Grouping
Group sessions by project, date, or custom criteria.

- `/sessions` page gains group-by dropdown: None, By Project, By Date, By Model
- Grouped view: collapsible sections with group name, session count, total cost
- Each group header shows aggregate stats
- Remembers last selected grouping in localStorage

**Files:** update `src/app/sessions/page.tsx`.

### Pillar 3: Cost Forecasting
Predict future costs based on historical patterns.

- Cost forecast widget on `/costs` page
- Projects next 7 days and next 30 days of costs based on rolling average
- Shows confidence range (min/max based on standard deviation)
- Line chart: actual cost line + dashed forecast line with shaded confidence band
- Warning if forecast exceeds monthly budget

**Files:** `src/components/cost-forecast.tsx` (new), update costs page to include the component.
