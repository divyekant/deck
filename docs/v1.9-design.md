# Deck v1.9 — Design

Session insights, focus mode, and command history.

## Pillars

### Pillar 1: Session Insights
AI-style insights derived from session patterns (no actual AI — heuristic-based).

- `/insights` page showing automatically generated insights
- Insight types: "Most productive day of week", "Peak coding hours", "Favorite model", "Longest streak", "Most expensive project", "Most efficient sessions (high cache hit rate)", "Busiest project this week"
- Each insight: icon, title, description with data, chart or stat
- Insights computed from session data on the server
- Refreshes on page load

**Files:** `src/app/insights/page.tsx` (new), `src/app/api/insights/route.ts` (new), update sidebar, update command palette.

### Pillar 2: Focus Mode
Distraction-free view showing only the current active session.

- `/focus` page that shows the most recent active (or last completed) session
- Minimal UI: just the conversation with large font
- Session stats bar at top (duration, cost, tokens)
- Auto-refreshes if session is still running
- Keyboard shortcut: Ctrl+Shift+F to toggle focus mode
- Back button to return to normal view

**Files:** `src/app/focus/page.tsx` (new), update keyboard-nav component.

### Pillar 3: Command History
Browse all terminal commands executed across Claude Code sessions.

- `/commands` page showing extracted terminal commands
- Groups by session with timestamps
- Search/filter commands by keyword
- Copy command button
- Shows command output preview (first 3 lines)
- Useful for finding "how did I do X last time?"

**Files:** `src/app/commands/page.tsx` (new), `src/app/api/commands/route.ts` (new), update sidebar, update command palette.
